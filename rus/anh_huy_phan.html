Проведение исследований по следующим основным проектам

  • Тензорные разложения (TDs) и новые приложения: работа с наиболее сложными задачами в тензорных разложениях, например, с высокой вычислительной стоимостью алгоритмов, методом оптимизации второго порядка, мерой производительности алгоритмов, массивным тензорным разложением, тензорной дефляцией.

  • Разработка надежных алгоритмов с низкими вычислительными затратами для различных малоранговых тензорных разложений крупномасштабных данных, включая Candecomp/Parafac, тензорные разложения Таккера, низкоранговая тензорная развертка, диагонализицая тензора, тензорное разложение Кронекера, тензорная дефляция, извлечение характеристик для многоканальных данных.

  • Разработка современных алгоритмов для тензорного анализа, особенно для тензорного разложения Candecomp/Parafac.

  • Слепое разделение сигналов. Разработка алгоритмов для слепого разделения сигналов, основанных на независимом компонентном анализе, неотрицательной матричной факторизации и тензорного разложения сети. Методы состоят из двух этапов: тензоризация сигналов смеси для получения тензоров, низко-ранговое тензорное разложение для извлечения скрытых сигналов

  • Нейрокомпьютерный интерфейс (НКИ): применение тензорного разложения для извлечения соответствующих характеристик для сигналов ЭЭГ в системе НКИ.

  • Идентификация нелинейной системы на основе многомерной полиномиальной регрессии, моделей типа Вольтерра, в которых параметры представлены в формате Тензорной сети. Модели могут быть применены для регрессии, дискриминантного анализа, метода опорных векторов, глубокого обучения, слияния данных и комбинации характеристик.

  • Другие исследования:
    • Раннее выявление болезни Альцгеймера
    • Развертка данных крупномасштабной визуализации кальция
    • Кодирование граней по компонентам с ограничениями сложности
